Plugin for adding a widget for The Visual Verse of the Day at http://visualverse.thecreationspeaks.com.

== Description ==

This plugin adds a widget for [The Visual Verse of the Day](http://visualverse.thecreationspeaks.com/).
It dynamically grabs the latest image and verse reference.

== Installation ==

1. Find the plugin in the Wordpress plugin gallery and click "Install Now".
2. In the Wordpress dashboard, go to Appearance > Widgets.
3. Find the new Visual Verse of the Day widget and click on it.
4. Choose the desired widget area and click "Add Widget".
5. Find the list of widgets for that widget area and drag them into the desired order.

== Frequently Asked Questions ==

= What is the "width" setting for? =

If you don't want the widget to be the full width of the widget area, set a width in CSS style (like 200px; or 80%;).

= Can I style it more? =

Sure, find and edit the file wp-content/plugins/vv-widget/css/vv-widget-styles.css.
